# Prompt Dataset Builder 1.1.0

Android-приложение для ручной подготовки текстового датасета `command / input / output` по изображениям Civitai.

## Главное исправление 1.1.0

Публичный REST `/api/v1/images` больше не используется как источник промптов. Он может вернуть изображения, но не вернуть `meta.prompt`.

Новая схема:

1. Лента загружается через `image.getInfinite` с `civita.red`.
2. Для каждого ID отдельно вызывается `image.getGenerationData`.
3. Изображение показывается только после получения открытого prompt.
4. При ошибке `civita.red` автоматически используется `civitai.com`.
5. REST `/api/v1/images` в сетевом коде отсутствует.

## Возможности

- исходный prompt скрыт до ввода пользовательского описания;
- формат записи: `command`, `input`, `output`;
- изображения не сохраняются в датасет и не кешируются на диске;
- обработанные ID хранятся локально;
- API-ключ хранится через Android Keystore;
- NSFW включается только с API-ключом и подтверждением 18+;
- экспорт JSONL;
- сборка APK через GitHub Actions.
