# Prompt Dataset Builder Clean 1.0.0

Полностью новая Android-версия приложения для ручного формирования датасета из трёх полей:

```json
{"command":"...","input":"...","output":"..."}
```

## Главное отличие от старых версий

Приложение больше не обращается к `civita.red`. Этот сайт возвращал HTML-страницу вместо JSON и из-за этого возникала ошибка `<!DOCTYPE ... cannot be converted to JSONObject`.

Источник теперь один и зафиксирован в коде:

```text
https://civitai.com/api/v1/images
```

Используется официальный Civitai REST API.

## Возможности

- изображения загружаются только для просмотра;
- дисковый кэш изображений отключён;
- исходный промпт скрыт до ввода собственного описания;
- промпт можно отредактировать перед сохранением;
- обработанные изображения больше не показываются;
- API-ключ хранится локально через Android Keystore;
- NSFW включается только с API-ключом и подтверждением 18+;
- экспорт JSONL с полями `command`, `input`, `output`;
- встроенная диагностика HTTP-ответа без показа API-ключа;
- автоматическая сборка APK через GitHub Actions.

## Сборка

Workflow `Android CI Clean 1.0.0` запускается после push в `main` или `master`.
Готовый APK находится в artifact:

```text
PromptDatasetBuilder-Clean-1.0.0-debug
```

## Требования

- Android 8.0 или новее;
- интернет;
- API-ключ Civitai для NSFW-контента.
