УСТАНОВКА PROMPT DATASET BUILDER 1.1.0 В CODESPACES

1. Загрузите PromptDatasetBuilder-1.1.0.zip в корень репозитория через сайт GitHub.
2. Откройте Codespaces -> Terminal -> New Terminal.
3. Вставьте весь блок:

cd /workspaces/PromptDatasetBuilder

git fetch origin
git reset --hard origin/main

ZIP="PromptDatasetBuilder-1.1.0.zip"
test -f "$ZIP" || { echo "ZIP не найден"; exit 1; }

rm -rf /tmp/pdb-1.1.0
mkdir -p /tmp/pdb-1.1.0
unzip -q "$ZIP" -d /tmp/pdb-1.1.0

find . -mindepth 1 -maxdepth 1 \
  ! -name ".git" \
  ! -name "$ZIP" \
  -exec rm -rf {} +

cp -a /tmp/pdb-1.1.0/. .
rm -rf /tmp/pdb-1.1.0
rm -f "$ZIP"

python3 scripts/verify_project.py
git add -A
git status
git commit -m "Полная установка Prompt Dataset Builder 1.1.0"
git push origin main

4. Откройте GitHub -> Actions -> Android CI 1.1.0.
5. После зелёной сборки скачайте artifact PromptDatasetBuilder-1.1.0-debug.
6. Установите PromptDatasetBuilder-1.1.0-debug.apk.
